# Css Tricks
## Fade effect
To add fade effect in canvas use\
fillStyle = (* ,* ,*,a<1)\
fillrect()
## Gravity
To add gravity to a particle incress velocity y over time.\
dy = dy + gravity

## Friction 
To add friction to a particle multiply velocity to a number < 1.\
dx = dx * 0.99\
dy = dy * 0.99
